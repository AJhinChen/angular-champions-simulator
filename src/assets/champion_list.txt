1.获取英雄姓名和图片名称列表
REQUEST URL
https://na1.api.riotgames.com/lol/static-data/v3/champions?locale=zh_CN&tags=image&dataById=false

2.获取物品和图片的名称列表
REQUEST URL
https://na1.api.riotgames.com/lol/static-data/v3/items?locale=zh_CN&tags=image

3.获取符文和图片的名称列表
REQUEST URL
https://na1.api.riotgames.com/lol/static-data/v3/runes?locale=zh_CN&tags=image

4.获取召唤师技能和图片的名称列表
REQUEST URL
https://na1.api.riotgames.com/lol/static-data/v3/summoner-spells?locale=zh_CN&dataById=false
